**A. NỘI DUNG THỰC HIỆN (6 điểm)**

Căn cứ vào bài tập lớn (BTL) đã đăng ký, nhóm thực hiện các nội dung sau:
**1. Phân tích và Thiết kế hệ thống (Lý thuyết – 3 điểm)**

- **Thiết kế UML** :
  o Ít nhất 01 biểu đồ Use Case tổng quát.
  o Tối thiểu 03 Use Case chi tiết cho các chức năng chính.
  o Ít nhất 02 Sequence Diagram cho các chức năng quan trọng.
- **Đặc tả Use Case** :
  o Mô tả chi tiết luồng sự kiện chính và ngoại lệ.
  o Sử dụng mẫu đặc tả chuẩn (đã cung cấp).
- **Thiết kế giao diện (UI/UX)** :
  o Dùng Bootstrap để thiết kế giao diện responsive.
  o Yêu cầu có: Trang chủ, form nhập liệu có validation cơ bản, menu điều
  hướng, và dashboard quản trị.
  o Khuyến khích nhóm trình bày mockup trước khi code.
  **2. Cài đặt và Lập trình (Thực hành – 3 điểm)**
- **DataBase** : Cài đặt MySQL/PostgreSQL, tạo cấu trúc bảng, export file SQL.
- **Tầng dữ liệu (DAO)** : Triển khai kết nối JDBC, CRUD đầy đủ.
- **Tầng nghiệp vụ** : Khai báo Interface, cài đặt logic xử lý theo Use Case.
- **Kiểm thử** : Mỗi nhóm phải có ít nhất 02 test case minh họa (unit test hoặc
  integration test).
